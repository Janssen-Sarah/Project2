Portfolio in HTML/CSS

Een website bouwen in HTML en CSS

Dit project is de basis van mijn huidige website. Het is mijn eerste project in HTML/CSS. Voor dit project was het de bedoeling om een website uit te bouwen zonder gebruik van een framework zoals bijvoorbeeld Bootstrap. De website werd opgebouwd als desktopversie en is daarom niet volledig responsive. Voor de start van het opbouwen werd er een volleidge analyse uitgewerkt. In deze analyse wordt er een voorbereiding uitgewerkt, van inspiratie opzoeken tot mockups en een eigen huisstijl. Dit onderdeel vormt dan ook de basis van een goede website. Het volledige project is hier te bekijken. Gelieve er rekening mee te houden dat dit project de beste weergave heeft via desktop.
