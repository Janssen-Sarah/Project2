Groepsproject in Wordpress

Een website bouwen

Als onderdeel van de opleiding werkte ik mee aan een groepsproject. We maakten in een groepje met 4 personen een website voor een echt bedrijf. Bedoeling was om deze website op te bouwen in Wordpress, om zo ook met deze werkwijze vertrouwd te worden. Zowel de back-end als front-end werd volledig door ons opgemaakt. Er werden verschillende pagina's aangemaakt met ieders hun eigen doel. Zo was er een homepagina en een contactpagina, alsook een pagina waarop events werden bijgehouden.

Werken met Elementor

Om de website op te bouwen maakten we gebruik van de plugin Elementor. Deze pagebuilder maakt het mogelijk om op een eenvoudige manier elementen in Wordpress op te maken. Men kan meteen visueel zien hoe een pagina er zal uitzien en aanpassingen zijn met enkele klikken mogelijk. Ook kan er makkelijk gezorgd worden voor een mooie responsiviteit op de pagina's. Dit project was voor mij een eerste kennismaking met Wordpress en met het ontwerpen van websites. Ik leerde veel bij over de verschillende aspecten die aan bod komen bij het bouwen van een website. Het was dus een zeer fijn project om aan mee te mogen werken.
