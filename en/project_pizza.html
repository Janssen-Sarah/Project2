Project pizza

First project

My very first project meant an immediate leap of faith. As a newcomer to the IT world, and with my entry in the middle of the academic year, finding my way around was quite a challenge. The uncertainty and the unknown did not make it any easier. Nevertheless, I was determined to persevere and give it my all. As I practised more and more, I eventually found my way to create a nice first project. I created an SQL database for a fictitious pizza company. You can read more about this project below.

Introduction to SQL

This project involves converting an existing ERD diagram into an SQL database for a fictitious pizza company. The task involves translating the ERD diagram into an SQL script for creating the tables, specifying all elements in the data model, such as data type, FK/PK and NULL/NOT NULL constraints. The script first drops all the tables and then recreates them, so it can be run multiple times. I also created a schema to place the tables in. Furthermore, several records had to be filled in for each table, including the association entities. The INSERT-INTO statement is used to add these records. This project is a first hands-on experience in using SQL Server Management Studio and converting an ERD schema to a SQL database.
