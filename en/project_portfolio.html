Portfolio in HTML/CSS

Building a website in HTML and CSS

This project is the basis of my current website. It is my first project in HTML/CSS. For this project, the intention was to build out a website without using a framework such as Bootstrap, for example. The website was built as a desktop version and is therefore not fully responsive. Before starting the build, a full analysis was worked out. In this analysis a preparation is worked out, from finding inspiration to mockups and an own corporate identity. This part is therefore the basis of a good website. The full project can be viewed here. Please note that this project has the best viewing via desktop.
